# Advanced Scientific Calculator 📱

A robust, professional-grade Android scientific calculator built with Java and XML. This application features a custom-built mathematical expression parser capable of handling complex string-based equations without relying on external evaluation libraries.

## ✨ Features
* **Standard Operations:** Addition, Subtraction, Multiplication, Division, and Percentages (`%`).
* **Trigonometry:** `SIN`, `COS`, `TAN` (Calculated in Degrees).
* **Inverse Trigonometry:** `SIN⁻¹`, `COS⁻¹`, `TAN⁻¹`.
* **Advanced Mathematics:** Factorials (`N!`), Square Roots (`√`), Exponents (`X²`, `Xʸ`), Inverses (`1/X`), Logarithms (`LOG`, `LN`).
* **Constants:** Pi (`π`) and Euler's Number (`e`).
* **Custom Parser:** Safely parses and evaluates nested parentheses and order of operations using a recursive descent parsing algorithm.
* **Modern UI:** Sleek, high-contrast dark mode interface inspired by flagship native calculators.
